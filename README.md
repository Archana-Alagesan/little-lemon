## What I Built & Learned

This project demonstrates practical React skills applied to a real-world use case:

- **Component Architecture** ‚Äî Broke down the UI into reusable, single-responsibility components (Header, Nav, BookingForm, MenuCard, Footer, etc.)
- **State Management** ‚Äî Used `useState` and `useReducer` to manage form state and available time slots
- **React Router** ‚Äî Implemented multi-page navigation with `<Link>`, `<Routes>`, and `<Route>`
- **Controlled Components** ‚Äî Built a fully controlled booking form with validation logic
- **Responsive Design** ‚Äî Applied CSS Flexbox and Grid for a mobile-first, adaptive layout
- **Accessibility** ‚Äî Added semantic HTML5 elements and ARIA labels to ensure screen reader compatibility

## Getting Started

### Prerequisites
- Node.js v16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Archana-Alagesan/little-lemon.git

# Navigate into the project
cd little-lemon

# Install dependencies
npm install

# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
